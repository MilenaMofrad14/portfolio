Arrays

What is an array?

A structure consisting of values that have the same data type stored in the same block of memory. They can store strings, primitive data types, and even objects.

Terminology

Element : Individual locations in array
Index: Represent location in array. Written in square brackets after name of array (arr[1]) You can access any element in the array using its index.
Array Length: An array length is determined when it is initialized and it cannot be changed after.
2D arrays: Used when information needs to be stored in rows and columns.

Array Declaration and Initialization

Arrays are declared with their data type followed by square brackets. They can be initialized with values, and their values can later be changed.

An array can be passed into methods and returned from methods the way other data types are, the original array passed through will not be changed.

ArrayList

An ArrayList uses an array to store elements, but its size can be changed. Elements can be removed or added. The ArrayList works by keeping track of how many elements are currently in the array (size), and how the length of the array (capacity). When the list runs out of space, the capacity doubles.

Searching

Searching algorithms are used to find a specific value in an array. One of the searching algorithms is sequential search.

Sequential Search

Sequential search starts with the first element in the array, and checks if it is the targeted element, if it is not it continues. It loops through the array, checking each element, until the targeted element is found.

Sorting

Sorting algorithms are used to reorganize arrays in a specific order, often from least to greatest or greatest to least. There are many different sorting algorithms that vary in speed, effectiveness, and complexity to learn.

Selection Sort

This sorting algorithm creates a sorted and unsorted section of the array. It finds the maximum value of the unsorted section and moves it to the beginning of the unsorted section. This continues until the entire array is in the sorted section.
