Inheritance

A class can inherit methods and attributes from another class. The class that is inheriting is called the subclass, child, or derived class, the class that is being inherited from is called the superclass or parent. A derived class can have a derived class. This creates a class hierarchy in which any class that is higher is an ancestor, and any lower class is a descendant. The subclass will inherit all the public methods and attributes. Inheritance allows for more reusing of code.

Polymorphism

Multiple classes that are linked by inheritance may have methods that have the same name but do different things. This is called overriding. This allows for the same thing to be done in different ways in accordance to each class.

super() is used to call an overridden method or constructor in a superclass. this() calls the method in the current class.

Abstract Classes

Abstract classes cannot be called to create an object, and abstract methods are in abstract classes and they are empty. Abstract classes are used to provide a base for subclasses to inherit. The subclasses will define what the abstract methods will do.
